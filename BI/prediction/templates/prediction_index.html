Prediction with the SVR Model

This prediction uses the SVR Model.

SVR (Support Vector Regression) is a regression algorithm based on the same algorithms used by Support Vector Machines (SVM) to create classification models. Although there are some differences because the output of a regression is a real value and not a label. Which makes it very difficult to predict objective values. In the case of a regression there are infinite possibilities, compared to the limited number existing in classification problems.
